# Stack 1

Repositorio base del piloto *Desarrollo Distribuido con IA*. Clonar, abrir en Cursor y construir scripts del área con asistencia de IA.

## Requisitos

- Python 3.11+
- [Cursor](https://cursor.com)

## Instalación

```powershell
git clone <URL_DEL_REPO>
cd "Stack 1"
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

En Cursor: **File → Open Folder** y seleccionar la carpeta del proyecto.

## Uso

1. Leer [TEMPLATE.md](TEMPLATE.md) para entender la estructura, reglas y skills del proyecto.
2. Configurar rutas en `config.py`.
3. Crear scripts en `scripts/` con ayuda de Cursor.
